| Application Project  Title: | *KindWords: Daily Motivation Generator for Positivity and Mood Support* |
| ----- | :---- |
| **Project Description/ Executive Summary** | *KindWords is a simple Android app that I will develop. It provides quick random compliments, motivational quotes, or positive affirmations whenever you need a fast mood lift. You just tap one big button and it shows a new positive message from a list of over 100 that I will include in the app. You can also set a daily notification to remind you at a time of your choice. There’s an option to save your favorites so you can revisit them when things get tough. I built it solely for Android using Flutter because it’s quick to create and looks clean. The app consists of one main screen with a basic design and it includes a little animation when the message appears. It works completely offline, so there’s no need for internet, accounts, or any server components. This design makes it easy to build and ensures it runs smoothly. By the end, I’ll have a working prototype with all the main parts connected, some documents explaining everything and the code uploaded to GitHub. This app is meant to be a helpful tool, especially for students like us, who sometimes just need a quick pick-me-up during busy/stressful days.* |
| **Project Objectives** |  |
| ● *Specific  Develop a simple Android app called KindWords using Flutter that includes:  one-tap button to show a random positive message from a built-in list of 100+ quotes, option to set one daily notification time so a random quote is sent automatically, ability to save favorite quotes and view them in a separate list.*  ● *Measurable  The app must contain at least 100 unique quotes, show varied random messages (no immediate repeats), deliver the daily notification correctly at the chosen time, and keep saved favorites even after closing and reopening the app. All features will be tested and confirmed working 100% on an Android emulator and phone.*  ● *Achievable  This is realistic for a midterm because it only uses basic Flutter widgets and two small packages (flutter\_local\_notifications and shared\_preferences). No backend, internet or complicated features are needed. I can complete it in 3–5 weeks with the skills and laptop I already have.*  ● *Relevant  The project directly supports our application development and emerging technologies course. It practices mobile UI design, Android notifications with scheduling, local data storage, and building a simple wellness app—skills that match current IT/CS trends in 2026\.*  ● *Time-bound  Finish the working prototype (all main features connected and tested), prepare the GDD-style documentation, upload the code to GitHub, and submit the complete project by the midterm deadline this term.* |  |
